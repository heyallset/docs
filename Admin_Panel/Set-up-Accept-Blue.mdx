## Introduction

Setting up an API Key, PIN, and Tokenization is an essential part of integrating your Accept Blue account. Follow these steps to ensure your merchant account is properly configured for secure transactions.

## Create an API Key and PIN

<Steps>
  <Step title="Step 1">
    ## Navigate to Control Panel

    Log into your Accept Blue Dashboard, then go to the left-hand menu and click Control Panel.

    ![Control Panel Pn](/images/control-panel.png)
  </Step>
  <Step title="Step 2">
    ## Click on Source Management

    Inside the Control Panel, select Source Management to manage your API and token keys.

    ![Source Management Pn](/images/source-management.png)
  </Step>
  <Step title="Step 3">
    ## Click “\+ Create Key”

    Click the \+ Create Key button to begin setting up a new API key.

    ![Create Key Pn](/images/create-key.png)
  </Step>
  <Step title="Step 4">
    ## Enter the Merchant’s Name

    Input the merchant's name you'd like to associate with this key.

    ![Merchant Name Pn](/images/merchant-name.png)
  </Step>
  <Step title="Step 5">
    ## Create a PIN for Your Merchant

    Enter a secure PIN with a minimum of 4 digits.

    ![Api Pin Pn](/images/api-pin.png)

    <Note>
      This PIN will be entered into All Set under Accept Blue API Key PIN.
    </Note>
  </Step>
  <Step title="Step 6">
    ## Enter the Merchant’s Email

    Provide the merchant's contact email.

    ![Merchant Email Pn](/images/merchant-email.png)
  </Step>
  <Step title="Step 7">
    ## Enable All Commands

    Scroll down and ensure that all command checkboxes are selected.

    ![Enable All Commands Pn](/images/enable-all-commands.png)
  </Step>
  <Step title="Step 8">
    ## Click “Save” to Generate the API Key

    Click save to confirm and generate your API Key.

    ![Click Save Pn](/images/click-save.png)
  </Step>
  <Step title="Step 9">
    ## Copy the API Key

    Once the key is generated, copy the API Key generated. 

    ![Generated Api Key Pn](/images/generated-api-key.png)

    ### 
  </Step>
  <Step title="Step 10">
    ## Enter Your API Key & PIN in All Set

    ### In your All Set dashboard:

    Go to Settings → API Keys

    ![Settings Api Pn](/images/settings-api.png)

    Enter your API Key, and API Key PIN in the provided fields

    ![Key And Pin Pn](/images/key-and-pin.png)
  </Step>
</Steps>

## Create a Tokenization Key

<Steps>
  <Step title="Step 1">
    ## Click “\+ Create Key” Again

    Click \+ Create Key to begin creating a new key.

    ![Create Key Again Pn](/images/create-key-again.png)
  </Step>
  <Step title="Step 2">
    ## Change the Source Key Type

    From the Source Key Type dropdown, select Tokenization instead of API.

    ![Tokenization Pn](/images/tokenization.png)
  </Step>
  <Step title="Step 3">
    ## Enter the Merchant’s Name

    Input the merchant name again for this key.

    ![Token Merchant Name Pn](/images/token-merchant-name.png)
  </Step>
  <Step title="Step 4">
    ## Click Save

    Click Save to complete and generate your tokenization key.

    ![Save Tokenization Pn](/images/save-tokenization.png)
  </Step>
  <Step title="Step 5">
    ## Copy the Tokenization Key

    After saving, copy the Tokenization Key generated.

    ![Copy Tokenization Key Pn](/images/copy-tokenization-key.png)
  </Step>
  <Step title="Step 6">
    ## Enter Your Tokenization Key in All Set

    In your All Set dashboard, paste your new Tokenization Key into the Accept Blue Tokenization Key field.

    ![Tokenization Key Pn](/images/tokenization-key.png)
  </Step>
</Steps>
