## Introduction

All Set is designed to be fully mobile-friendly, giving you the freedom to manage your business anytime, anywhere. Easily add it to your home screen for quick access to your business settings, bookings, and updates—all right at your fingertips.



## For iPhone

1. Tap the **Share** icon (square with an arrow) at the bottom of the screen.

2. Scroll down and select **Add to Home Screen**.

3. Enter a name for the icon if you want, then tap **Add**.

4. The All Set icon will appear on your home screen for quick access.

## For Android

1. Open **Chrome** and navigate to the All Set website.

2. Tap the **Menu** icon (three dots) in the upper-right corner.

3. Select **Add to Home Screen**.

4. Enter a name for the shortcut if you want, then tap **Add**.

5. You’ll see the All Set icon on your home screen, giving you easy access anytime.
